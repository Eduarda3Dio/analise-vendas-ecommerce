# 📊 Análise Exploratória de Vendas com Python

## 📖 Sobre o projeto

Este projeto consiste em uma Análise Exploratória de Dados (EDA) utilizando Python. 
Para isso, foi criada uma base de dados fictícia contendo informações sobre vendas de um 
e-commerce, permitindo aplicar técnicas de manipulação, transformação e visualização 
de dados.
O objetivo é demonstrar habilidades em análise de dados utilizando as principais bibliotecas do ecossistema Python.

---

## 🎯 Objetivos

- Gerar uma base de dados fictícia de vendas.
- Realizar o tratamento e preparação dos dados.
- Criar novas variáveis para análise.
- Explorar os dados por meio de indicadores e gráficos.
- Identificar padrões de vendas e faturamento.

---

## 🛠 Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Análises realizadas

Durante o projeto foram desenvolvidas as seguintes análises:

- Produtos mais vendidos;
- Produtos com maior faturamento;
- Faturamento por categoria;
- Quantidade de pedidos por estado;
- Faturamento por estado;
- Evolução mensal do faturamento;
- Distribuição do status das entregas.

---

## 📈 Principais resultados

A análise permitiu observar diferenças entre volume de vendas e faturamento dos produtos, 
identificar categorias com maior representatividade financeira, comparar o desempenho entre 
estados e acompanhar a evolução do faturamento ao longo do período analisado.
